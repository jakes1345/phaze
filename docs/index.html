E2EE direct messages

Messages are encrypted on your device before they leave. The relay stores ciphertext and can't read them.

Voice & video calls

P2P WebRTC with TURN fallback so calls work behind NAT. Screen sharing on the web client.

Voice messages

Record and send voice notes from the web client. Stored encrypted, plays back inline.

Cross-device history

Set a Recovery PIN and your message history follows you to any device you sign into.

Chat features

Reactions, edits, deletes, @mentions, pinned messages, search, ⌘K palette, slash commands, dark mode.

Self-hostable

Run your own relay in Docker. Point clients at your WebSocket URL. Litestream-to-S3 backups when configured.

Quick start

The web client at phazechat.world/web is the easiest way to try Phaze — no install needed. Native apps are also available for Linux, Windows and Android (macOS/iOS deferred; the web client works great on every platform). After signing up, set a Recovery PIN in Settings → Backup & Devices so you can use the same account on multiple devices.

Run your own relay

docker run -p 8080:8080 -v nexus_data:/data \
  ghcr.io/jakes1345/skype7-reborn-nexus:latest

Or clone the repo and go run ./nexus_server. Point clients at your WebSocket URL (e.g. wss://your-host/ws).
