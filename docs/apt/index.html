Phaze APT Repository

Add the Phaze repo once and your system Update Manager will notify you whenever a new version ships.

Step 1 — Add the signing key
curl -fsSL https://apt.phazechat.world/apt/phaze.gpg \
  | sudo gpg --dearmor -o /etc/apt/keyrings/phaze.gpg
Step 2 — Add the repository
echo "deb [arch=amd64 signed-by=/etc/apt/keyrings/phaze.gpg] \
  https://apt.phazechat.world/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/phaze.list
Step 3 — Install
sudo apt update && sudo apt install phaze

Requires Ubuntu 22.04+ / Debian 12+ (amd64). After setup, Update Manager will show Phaze updates automatically.
For other platforms see phazechat.world.
